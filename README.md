## üîÑ Project Workflow / Steps

1. **Data Loading**

   * Imported raw CSV data into Python for analysis.

2. **Exploratory Data Analysis (EDA)**

   * Analyzed distributions, trends, and customer behavior patterns.
   * Identified missing values and data quality issues.

3. **Data Cleaning**

   * Handled missing and inconsistent values.
   * Standardized columns for analysis readiness.

4. **SQL Analysis (PostgreSQL)**

   * Loaded cleaned data into PostgreSQL.
   * Wrote SQL queries to answer key business questions such as:

     * Revenue trends
     * Customer segmentation
     * Product category performance

5. **Dashboard Development (Power BI)**

   * Built an interactive dashboard with KPIs and filters.
   * Visualized customer behavior, revenue, and category insights.

6. **Reporting & Presentation**

   * Created a structured analytical report.
   * Developed a business presentation using **Gamma** to communicate insights clearly.
